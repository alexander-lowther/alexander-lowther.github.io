School Projects

  • Windows Process Scheduler
  • A C program that launches processes locally & schedules each instance to the users desired order. We can display which processor is being used in real time, as well as choose which scheduling algorithm to use. This is done using Windows API's.
  • Photoshopping the Hard Way
  • "Draw" a checkerboard background. Flip, rotate, and blend images together using only Java & the images given. This is done by mathematically interpreting the added images pixels locations relative to our Framebuffer.
  • Custom Windows Launcher
  • A program written in C that prompts the user with a list of Windows programs to launch. It can be used to launch your favorite and most useful programs faster. Again using API's to aquire handles to each process.
  • Jave GUI Basics
  • This program creates shapes that can simply be dragged with mouse input. Simultaneously, we use the inheritance of MouseListener to collect movement data and display it to the user.
  • Fractal Animations
  • Using fractals and matrix algebra, create multiple animations that will grow linearly and form interesting animations/art. This is done using Java only and includes Pythagoras tree, Koch snowflake, and the Sierpinski triangle.
  • Real Location Submarine Game in IOS
  • Using a compass, shoot a torpedo at your friend (less than 10 miles away). Done using SwiftUI and Apple maps, it displays an animation when you've entered the correct degrees (0-360) between you and your friend. The user selects a trajectory which dictantes wether you've "hit" a friend with your torpedo launch.
